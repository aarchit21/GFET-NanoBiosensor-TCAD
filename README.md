# GFET Nano-Biosensor – TCAD Simulation (IEEE APSCON)

Simulation-Based Performance Study of BioGFET Sensors Using TCAD.
This repository contains supporting material for the IEEE APSCON poster.

## 🔬 Overview
This work presents a graphene field-effect transistor (GFET) nano-biosensor
functionalized with antibodies for antigen detection. Dirac voltage shift
(ΔVDirac) is used as the primary sensing metric.
